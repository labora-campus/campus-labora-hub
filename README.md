# Campus Labora Hub

Plataforma educativa para la gestión de bootcamps de Inteligencia Artificial y No-Code.

## Tecnologías

- **Frontend**: React, Vite, TypeScript
- **UI**: Tailwind CSS, Shadcn/ui, Lucide React
- **Backend/Data**: Supabase (Auth, Database, Storage)
- **State Management**: TanStack Query

## Instalación y Desarrollo

1.  Clonar el repositorio
2.  Instalar dependencias: `npm install`
3.  Configurar variables de entorno en `.env`
4.  Iniciar servidor de desarrollo: `npm run dev`

## Estructura del Proyecto

- `/src/pages`: Vistas principales (Admin/Student)
- `/src/components`: Componentes reutilizables
- `/src/services`: Lógica de comunicación con Supabase
- `/src/hooks`: Custom hooks para gestión de estado y datos
- Edit files directly within the Codespace and commit and push your changes once you're done.
